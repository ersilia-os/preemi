# PREEMI modeling

PREEMI dataset analysis and modeling at CIDRZ

This repository contains code to produce the models and figures related to the article "Machine Learning Algorithm for Prediction of Perinatal Mortality in Zambia" by Manasyan et al.

Note that code is made available but is not executable since source data needs to remain private. If you need access to the source data, please request access to [this folder](https://drive.google.com/drive/folders/11H7lK9H4MRFMVk65gkbvRgGeDOYWeKwF?usp=sharing).
